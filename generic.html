Building a Local RAG Engine for Secure Knowledge Retrieval

This project focused on designing a Retrieval-Augmented Generation system that runs entirely locally, enabling users to query private research documents without relying on external APIs. The architecture prioritizes data sovereignty, epistemic transparency, and long-term extensibility.

Problem

  • Sensitive documents could not leave the local environment
  • LLM responses needed grounding in verifiable sources
  • Retrieval accuracy and response quality had to be auditable

Approach

  • Hybrid retrieval combining dense embeddings and keyword search
  • Structured reasoning and belief reporting
  • Modular ingestion, retrieval, and generation pipeline

Results

  • Accurate, source-grounded responses
  • Fully offline-capable architecture
  • Improved trust and repeatability in AI-assisted research

Stack

Python, Local LLMs, Vector Databases, NLP Pipelines

CallSense: Turning Customer Calls into Actionable Intelligence

CallSense was built to analyze customer call data at scale, extracting sentiment, intent, and recurring behavioral patterns to support faster, data-backed decision-making.

Problem

  • High volume of unstructured call recordings
  • Manual reviews were slow and inconsistent
  • Key insights were often missed or delayed

Approach

  • Speech-to-text preprocessing
  • NLP-based sentiment and intent extraction
  • Analytics layer for aggregating insights

Results

  • Faster insight generation
  • Reduced manual analysis overhead
  • Clear visibility into customer behavior trends

Stack

Python, NLP Models, Data Pipelines, Streamlit
